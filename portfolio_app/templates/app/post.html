Bank Loan Document Verification system Management

This was a project I started back in November 2021 when I first learned python. I was the sole creator/developer for the first year of the project before I was able to hire help.

The database stored sample data for the entire Project and provided an interface for bank Employees and admins to view, add and send data to users.

bank statement would be generated by the system in PDF form and automatically sent to the users along with an interface for customers to be able to login with customer id and view.

Customers had the ability to view all their data with a dynamic search form, trend data, add notes to reports and even map their samples' physical location.

I worked on this system for 1 years before delivering this to the client.

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - Angular
  • - Postgres
  • - Mongodb
  • - Google Maps API
  • - XML2PDF - PDF generator
